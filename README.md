Quick Start
-----------

Prerequisite: nodejs, LUA 5.3, VSCode

1) Build Project

```
npm install
npm run build
```

2) Compile test.ts

create file test.ts

```TypeScript
class Class1 {
  constructor(private i: number) {
  }

  public show() {
    console.log(this.i);
  }
}

let c = new Class1(1);
c.show();
```

```
node __out/main.js test.ts
```

Now you have test.luabc

3) Run it.

```
lua test.luabc
```

Result:
```
1
```

Enjoy it.
